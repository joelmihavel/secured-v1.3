# Fonts

Place **Zin Display Condensed** here for headline typography.

1. Download the OTF from [Befonts](https://befonts.com/zin-display-font.html) (or [direct download](https://befonts.com/downfile/2ff64d9380a13f38e91fb8425fae81d4.149347)).
2. Save the file as `ZinDisplayCondensed-Demo.otf` in this folder.

The app will load it via `@font-face` in `src/app/globals.css`.
